## Technologies Used

- Frontend:
  - React
  - Vite
  - Tailwind CSS
  - PostCSS

- Backend:
  - Node.js
  - Express.js 
  - Database (MongoDB)

## Getting Started

### Prerequisites

- Node.js (latest stable version)
- npm 
### Installation

1. Clone the repository
https://github.com/Sagarshrestha2301/WasteManagement

2. Frontend Setup

cd Waste-Management
npm install


3. Backend Setup
cd backend
npm install

### Running the Application

1. Start the frontend
npm run dev

2. Start the backend
npm start
